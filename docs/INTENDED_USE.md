# Intended Use

NexusOS is an **enthusiast / engineering-preview** Linux distribution. It is
intended for evaluation, development, and experimentation — not as a certified or
supported production operating system.

## Appropriate use

- Trying Linux on Mac (Apple Silicon dual-boot, UTM) or Windows (WSL2)
- Learning about Debian-based desktops and the Asahi platform stack
- Contributing to or building the project from source

## Not recommended

- Sole OS on hardware you cannot afford to recover or repartition
- Production servers, regulated data, or environments requiring vendor support
- Redistribution of Apple Silicon / firmware-dependent **release binaries** (see [REDISTRIBUTION_POLICY.md](REDISTRIBUTION_POLICY.md))
